# Payly Mobile

Исправленный мобильный клиент на Expo / React Native для Android и iOS.

## Что исправлено

- приложение больше не должно падать, если backend недоступен;
- добавлен демо-режим без сервера — можно сразу открыть приложение в Android Emulator и проверить UI;
- добавлен таймаут и нормальные сообщения об ошибках сети;
- сохранена работа с backend, если он запущен;
- для Android сборки включён cleartext HTTP, чтобы `http://10.0.2.2:8000` работал в эмуляторе;
- new architecture отключена для более стабильного запуска на Expo SDK 53.

## Быстрый запуск

```bash
cd mobile
npm install
npx expo start
```

Потом нажмите `a`, чтобы открыть Android Emulator.

## Android Emulator backend URL

Для Android эмулятора обычно используется:

```bash
http://10.0.2.2:8000
```

Если backend не запущен, просто нажмите в приложении:

**Запустить демо без backend**

## Если нужен backend

В корне проекта:

```bash
pip install -r requirements.txt
uvicorn backend.main:app --reload
```

После этого в мобильном приложении можно:

- зарегистрироваться / войти;
- смотреть подписки;
- добавлять и удалять подписки;
- подключать Gmail / Mail.ru и импортировать найденные подписки.
